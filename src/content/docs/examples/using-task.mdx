# Using the Task Component

This example shows how to properly pass the guide number from frontmatter to Task components.

## Basic Usage

<Task title="Create a new repository" number={1} guideNumber={guideNumber}>
  First, create a new repository:

  ```bash
  mkdir my-project
  cd my-project
  git init
  ```
</Task>

<Task title="Add project files" number={2} guideNumber={guideNumber}>
  Next, add some initial files to your project:

  ```bash
  touch README.md
  echo "# My Project" > README.md
  ```
</Task>

## Manual Override

<Task title="Special case" number={3} guideNumber="7">
